## вњЏпёЏ Content Updates

### Change bio, tagline, or promo card
Edit `data/content.json`

**Key fields:**
- `name` вЂ” Your name (used in header, footer, hero)
- `tagline` вЂ” Short line shown in hero
- `heroDesc` вЂ” One-line description under your name
- `bio` вЂ” Array of paragraphs (supports inline HTML like `<strong>`)
- `tags` вЂ” Genre/keyword tags shown in bio section
- `socials` вЂ” List of `{ "platform": "...", "url": "..." }` objects
- `promo` вЂ” The promoted card on the Promo tab:
  - `type` вЂ” `"event"`, `"mix"`, `"note"`, or `"photo"`
  - `title`, `subtitle`, `date`, `image`, `link`, `cta`, `note`

### Add or remove events
Edit `data/events.json`

Each event:
```json
{
  "title": "SHOW NAME",
  "venue": "Venue Name",
  "city": "City, Country",
  "date": "2025-06-21",
  "ticketLink": "https://...",
  "note": "Optional short note",
  "past": false
}
```
Set `"past": true` to move to the Past Shows section.
Events with a date in the past are automatically shown as past regardless of this flag.

### Add music embeds
Edit `data/music.json` в†’ `embeds` array.

Each embed:
```json
{
  "platform": "Mixcloud",
  "title": "My Mix Title",
  "iframe": "<iframe ...></iframe>"
}
```
Copy the embed iframe code from Mixcloud/SoundCloud/Bandcamp share options.

### Add local audio files
1. Copy `.mp3` files to `assets/local-music/`
2. Add to `data/music.json` в†’ `localFiles`:
```json
{ "title": "Mix Name", "file": "assets/local-music/your-file.mp3" }
```

## рџ–јпёЏ Gallery Updates

1. Copy image files into the `gallery/` folder (any filename)
2. Run: `node update-gallery.js`
3. Commit & push

The site displays images in **random order** (reshuffled on each page load).

Supported formats: `.jpg` `.jpeg` `.png` `.gif` `.webp` `.avif`

## рџЋЁ Design Customisation

All design tokens are CSS custom properties at the top of `style.css`:

```css
:root {
  --accent: #e8ff00;        /* Main accent colour вЂ” change for a whole new vibe */
  --bg:     #0a0a0a;        /* Page background */
  --text:   #f0f0f0;        /* Main text */
  /* ... */
}
```

**To change the colour scheme:** edit `--accent` and related variables.  
**To change fonts:** update the Google Fonts `<link>` in `index.html` and the `--font-*` variables in `style.css`.

| Function | Section |
|---|---|
